# jibber-jabber-api


## Running Postgres DB with PG-Admin

First create network 

>`docker network create jib-jab-postgres`

Then create DB

>`docker-compose up --no-start`

Start the DB

>`docker-compose start`

#### PG-Admin Info
* user: admin
* password: admin
* port: 5050

#### PG DB info
* user: admin
* password: admin
* port: 5432
